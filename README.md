# clock-web
# ⏰ Modern Clock Website

A clean and interactive clock web application built using **HTML, CSS, and JavaScript**. This project features both a **digital clock** and an **analog clock**, along with a **dark mode toggle** for better user experience.

---

## 🚀 Features

* 🕒 Real-time **Digital Clock**
* 🕰️ Smooth **Analog Clock with moving hands**
* 🌙 **Dark Mode Toggle**
* 📱 Responsive and centered layout
* ⚡ Lightweight and fast (no external libraries)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Functionality and real-time updates

---

## ⚙️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Shreyas-D-M/clock-web.git
```

2. Navigate into the project folder:

```bash
cd clock-website
```

3. Open `index.html` in your browser.
